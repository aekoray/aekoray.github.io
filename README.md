# aekoray.github.io
Site personnel Bio Link style Bento avec effet Glassmorphism. 🌑

🚀 Fonctionnalités

    Design Glassmorphism : Effet de verre dépoli avec backdrop-filter.

    Bento Layout : Organisation en blocs modernes et responsives.

    Tab Animation : Titre du navigateur dynamique qui simule une machine à écrire (@aekoray).

    Lucide Icons : Intégration d'icônes vectorielles légères.

🛠️ Stack Technique

    HTML5 / CSS3 : Pas de framework lourd, juste du pur style avec des variables CSS (:root).

    JavaScript (Vanilla) : Logique simple pour l'animation du titre et l'initialisation des icônes.

    Lucide.js : Pour le rendu des icônes.

📸 Aperçu

    "Vous prenez 10% et vous en faites une généralité"

Le site s'adapte automatiquement à la taille de l'écran (Mobile-first design).
